1. Sustainable Waste Management System

SDG Goal
11
Target
11.6
Indicator
11.6.1
Problem
Efficiently route waste collection vehicles to minimize time and fuel usage while covering all city areas.
Algorithm
Dijkstra's Algorithm
Implementation Details
The graph was implemented using an adjacency list to represent city roads.
The Heap Data Structure (Min-Heap) was used for efficient execution.
Algorithm Efficiency: O(ElogV), where E = edges and V = nodes.

Target

Indicator
8.9.1
Problem
Suggest optimal eco-friendly travel itineraries while minimizing travel costs and ensuring heritage site preservation.
Algorithm
Knapsack Algorithm
Implementation Details
Graph Representation: Costs and cultural significance assigned to destinations.
Dynamic Programming: Used to solve the 0/1 Knapsack problem.
Constraints: Traveler's budget and time.
